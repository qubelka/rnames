RNames - Help - Instructions

Stepwise Data Entry

  1. Enter new reference
    1. Check if reference is already in database
      • Open: Main Menu -> Manage References -> and search for reference
    2. If not found, enter the new reference at RNames
      • Open: Main Menu -> Manage References -> Press: Add
    3. Enter new reference details at Zotero.org
      • Login to Zotero.org
      • Enter reference manually at https://www.zotero.org/groups/rnames/, or enter automatically from RIS.file, Google Scholar etc via Zotero button in Firefox /Chrome extension (see here for instructions)
    4. Copy permanent reference link from Zotero.org to RNames
      • copy Zotero reference (= Item) link and paste into “http://“ field in RNnames Reference entry form (The link should look like: https://www.zotero.org/groups/rnames/items/itemKey/5XU8IG6K)
    5. Press: Save
  2. Enter all new stratigraphic unit(s) contained in the reference
    1. open new parallel browser window with Manage Structured Names page -> Add
    2. enter new Name
      • a Name is something like “Saint Paul” , “Rockdale Run”, “Kullsberg Limestone”, “Cahabagnathus sweeti” , or a number like “453.7”
    3. select the correct Qualifier
    4. select the appropriate Location or enter a new one
      • currently locations are relatively deliberately entered as regions, counties or countries, do not go into much detail here, do not enter places, such as cities, villages etc, but regions
    5. Press: Save
  3. Enter referenced relations between stratigraphic units
    1. open Manage Referenced Data -> select the reference and click the magnifier -> this opens a new window and there you can add new relations by clicking the add button
    2. switch the "belongs to" button on "yes" in case the two structured names are related as group and sub-group: e.g. the Fort Cassin Formation belongs to the Beekmantown Group, or the alobatus conodont_sub_zone is a subzone of the anserinus condodont-zone..
